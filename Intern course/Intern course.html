Курс підготовки до InterLink inCamp

About cours:

This is a basic training course for a full stack web developer. After completing this course, you will be able to become an intern at InterLink.

Before passing the course, it is advisable to have:

  • Basic programming skills
    • control structures
      • assigning variables
      • branching
      • cycles
      • functions
    • data structures
      • arrays
      • associative arrays (map, hash, etc.)
  • Desire and ability to perform practical tasks
